SUGERENCIA DE TIPOS•Python es un lenguaje de tipado dinámico. Esto significa que el intérprete de Python realiza la comprobación de tipos sólo durante la ejecución del código, y que el tipo de una variable puede cambiar a lo largo de su vida.•Otro término que se utiliza a menudo cuando se habla de Python es ducktyping. Este apodo proviene de la frase "si camina como un pato y grazna como un pato, entonces debe ser un pato".•Ducktypinges un concepto relacionado con la tipificación dinámica, en la que el tipo o la clase de un objeto es menos importante que los métodos que define. Con el tipado dinámico no se comprueban los tipos. En su lugar, se comprueba la presencia de un determinado método o atributo.•La sugerencia de tipos (Typehinting), que introdujo en Python 3.5, es una solución formal para indicar estáticamente el tipo de un valor dentro del código Python.•He aquí un ejemplo de cómo añadir información de tipo a una función:defsaludar(nombre: str) →str:return"Hola, " + nombre
